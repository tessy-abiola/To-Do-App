#  Todo List App

A modern Todo List app built with HTML, CSS, and JavaScript, featuring task management, local storage, and a clean responsive UI.

##  Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Data saved using localStorage
- Responsive design
